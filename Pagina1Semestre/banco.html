Banco de Dados


Modelagem e SQL


Durante a disciplina de Banco de Dados 1, nos aprofundamos na modelagem de bancos de dados e na manipulação de dados utilizando a linguagem SQL (Structured Query Language). Através da modelagem, aprendemos a projetar esquemas de banco de dados eficientes e organizados, visando atender aos requisitos específicos de uma aplicação.


Além disso, exploramos as operações básicas de manipulação de dados em SQL, como inserção, consulta, atualização e exclusão. Comandos como SELECT, INSERT, UPDATE e DELETE foram utilizados para recuperar informações, adicionar novos registros, atualizar valores existentes e remover dados de tabelas. Também aprendemos sobre a linguagem de definição de dados (DDL), que permite a criação e modificação da estrutura dos bancos de dados.


Durante as aulas práticas, tivemos a oportunidade de criar e manipular bancos de dados em SQL, aplicando os conceitos aprendidos. Essas atividades nos permitiram adquirir experiência prática na utilização da linguagem SQL e entender como as operações de banco de dados são executadas. Através dessas atividades, desenvolvemos a capacidade de projetar e implementar bancos de dados eficientes, aplicando a abstração e seguindo boas práticas de modelagem.
