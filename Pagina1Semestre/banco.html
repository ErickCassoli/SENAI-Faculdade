Banco de Dados


Modelagem e SQL


Durante a disciplina de Banco de Dados 1, nos aprofundamos na modelagem de bancos de dados e na manipulação de dados utilizando a linguagem SQL (Structured Query Language). Através da modelagem, aprendemos a projetar esquemas de banco de dados eficientes e organizados, visando atender aos requisitos específicos de uma aplicação.


Além disso, exploramos as operações básicas de manipulação de dados em SQL, como inserção, consulta, atualização e exclusão. Comandos como SELECT, INSERT, UPDATE e DELETE foram utilizados para recuperar informações, adicionar novos registros, atualizar valores existentes e remover dados de tabelas. Também aprendemos sobre a linguagem de definição de dados (DDL), que permite a criação e modificação da estrutura dos bancos de dados.


Durante as aulas práticas, tivemos a oportunidade de criar e manipular bancos de dados em SQL, aplicando os conceitos aprendidos. Essas atividades nos permitiram adquirir experiência prática na utilização da linguagem SQL e entender como as operações de banco de dados são executadas. Através dessas atividades, desenvolvemos a capacidade de projetar e implementar bancos de dados eficientes, aplicando a abstração e seguindo boas práticas de modelagem.


Tópicos da disciplina de Banco de Dados 1


Modelagem de Bancos de Dados

Neste tópico, nos aprofundamos na modelagem de bancos de dados. Aprendemos a projetar esquemas de banco de dados eficientes e organizados, visando atender aos requisitos específicos de uma aplicação. Exploramos técnicas como o diagrama Entidade-Relacionamento (ER), identificação de entidades, relacionamentos, atributos e restrições. Compreendemos a importância da modelagem adequada para garantir a integridade e a eficiência dos dados armazenados.


Manipulação de Dados com SQL

Neste tópico, exploramos as operações básicas de manipulação de dados utilizando a linguagem SQL (Structured Query Language). Aprendemos comandos como SELECT, INSERT, UPDATE e DELETE para realizar consultas, inserir novos registros, atualizar valores existentes e excluir dados de tabelas. Compreendemos como a linguagem SQL permite a interação com os bancos de dados, fornecendo uma poderosa ferramenta para a manipulação e recuperação de informações.


Linguagem de Definição de Dados (DDL)

Durante o curso, também aprendemos sobre a linguagem de definição de dados (DDL) em SQL. Com essa linguagem, podemos criar e modificar a estrutura dos bancos de dados. Exploramos comandos como CREATE TABLE, ALTER TABLE e DROP TABLE, que nos permitem criar, alterar e excluir tabelas, colunas e restrições. Compreendemos como a DDL é utilizada para definir a estrutura e as características dos elementos do banco de dados.


Aulas Práticas de SQL

Durante as aulas práticas, tivemos a oportunidade de aplicar os conceitos aprendidos na manipulação de bancos de dados utilizando a linguagem SQL. Realizamos atividades práticas de criação, inserção, consulta, atualização e exclusão de dados, colocando em prática os comandos e técnicas aprendidos. Essas atividades nos proporcionaram experiência prática na utilização da linguagem SQL e nos ajudaram a compreender como as operações de banco de dados são executadas.


Normalização de Banco de Dados

A normalização de banco de dados é um processo importante para garantir a integridade e a eficiência dos dados armazenados. Através da normalização, aprendemos a eliminar redundâncias e inconsistências nos dados, organizando as tabelas de forma estruturada. Exploramos conceitos como as formas normais (1ª, 2ª e 3ª forma normal) e as dependências funcionais, que nos ajudam a identificar e corrigir problemas de estruturação dos dados. Compreendemos como a normalização contribui para a eficiência das consultas e a manutenção adequada do banco de dados.
