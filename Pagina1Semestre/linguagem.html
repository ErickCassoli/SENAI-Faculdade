Projetos


Currículo

Nosso primeiro projeto foi um currículo básico com html e css, basicamente foi para averiguarmos o nível da sala em geral em relação aos conhecimentos de programação. No meu currículo eu usei do meu conhecimento que ja tinha e das aulas dadas no começo do curso.

Exercícios java script

Nessa etapa do curso a turma executou diversos exercícios de JavaScript para aprendermos logica e a sintaxe do mesmo. Esses exercícios de fixação serviram de base para os projetos que fizemos em seguida.

Calculadora de salario

Nesse projeto a gente trabalhou em squads para criar uma site com a aplicação de realizar um calculo para saber o salario liquido, ja descontando os valores de impostos.

Transforme code

Ja esse projeto foi um complemento para os exercícios de javascript. A demanda foi que precisávamos de uma plataforma estilizada para testar os exercícios, então inicialmente foram escolhidos alguns alunos (incluindo eu) para que eles elaborassem essa plataforma, a partir desse original a sala teve que elaborar outros modelos aprimorados.
