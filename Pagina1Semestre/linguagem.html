Linguagem de Programação


TECNOLOGIAS: HTML, CSS e JavaScript


Durante o estudo da disciplina de Linguagem de Programação, foram abordadas várias linguagens essenciais no desenvolvimento web, incluindo HTML, CSS e JavaScript. HTML (HyperText Markup Language) foi explorada como a linguagem de marcação que permite estruturar e organizar o conteúdo de uma página web, definindo elementos como cabeçalhos, parágrafos, imagens e links. CSS (Cascading Style Sheets) foi estudada como a linguagem responsável pela estilização e formatação dos elementos HTML, permitindo definir cores, fontes, layout e outros aspectos visuais. Por fim, JavaScript foi introduzido como uma linguagem de programação que permite a interatividade e a manipulação de elementos em uma página web, adicionando comportamentos dinâmicos, validações de formulários e interações com o usuário.

Além das linguagens, também foi abordada a lógica de programação, que é fundamental para a construção de algoritmos e a solução de problemas computacionais. Aprendemos a estruturar programas utilizando estruturas de controle, como condicionais (if-else) e loops (for, while), para criar fluxos lógicos de execução. Também foram explorados conceitos como variáveis, operadores, funções, fornecendo as bases necessárias para o desenvolvimento de programas mais complexos. Além disso, foram abordados conceitos como o DOM (Document Object Model), que representa a estrutura hierárquica dos elementos de uma página web e permite a manipulação desses elementos por meio do JavaScript. A importância da semântica no desenvolvimento web também foi enfatizada, compreendendo a utilização adequada dos elementos HTML para conferir significado e estrutura correta ao conteúdo, proporcionando uma melhor acessibilidade.


Tópicos da disciplina de Linguagem de Programação


HTML (HyperText Markup Language)

HTML é uma linguagem de marcação utilizada para estruturar e organizar o conteúdo de uma página web. Ele fornece um conjunto de elementos que descrevem a semântica e a estrutura do documento, como cabeçalhos, parágrafos, listas, imagens e links. Ao utilizar esses elementos, podemos definir a hierarquia e a disposição do conteúdo na página, facilitando a compreensão e a formatação correta.


CSS (Cascading Style Sheets)

CSS é uma linguagem de estilo que permite controlar a aparência e o layout dos elementos HTML. Com o CSS, podemos definir cores, fontes, margens, preenchimentos, posicionamento e outros atributos visuais. Ele permite separar a estrutura e o conteúdo do documento da sua apresentação visual, tornando mais fácil aplicar estilos consistentes e personalizados em todas as páginas do site.


JavaScript

JavaScript é uma linguagem de programação que permite adicionar interatividade e comportamentos dinâmicos às páginas web. Com o JavaScript, podemos manipular elementos HTML, responder a eventos, realizar validações de formulários, criar animações, fazer requisições assíncronas ao servidor e muito mais. Ele é executado no navegador do usuário e é amplamente utilizado para criar experiências interativas e responsivas na web.


Lógica de programação

A lógica de programação é um conjunto de princípios e técnicas utilizados para resolver problemas computacionais. Ela envolve a estruturação de programas utilizando estruturas de controle, como condicionais (if-else) e loops (for, while), para criar fluxos lógicos de execução. Além disso, a lógica de programação inclui o uso de variáveis para armazenar dados, operadores para realizar cálculos e manipulações, e funções para organizar e reutilizar blocos de código.


DOM (Document Object Model)

O DOM é uma representação em memória dos elementos de uma página web, organizados em uma estrutura hierárquica. Ele permite que o JavaScript acesse, manipule e modifique dinamicamente os elementos e o conteúdo da página. Com o DOM, podemos adicionar, remover ou alterar elementos, modificar estilos, manipular eventos e realizar outras ações para criar interatividade e dinamismo nas páginas web.


Semântica no desenvolvimento web

A semântica no desenvolvimento web envolve o uso adequado dos elementos HTML para conferir significado e estrutura correta ao conteúdo. Isso significa escolher os elementos HTML apropriados para representar o tipo de conteúdo que está sendo apresentado, como usar tags de cabeçalho para títulos, tags de parágrafo para texto descritivo, tags de lista para itens relacionados, entre outros. A semântica adequada ajuda os mecanismos de busca a entenderem melhor o conteúdo da página e também melhora a acessibilidade para pessoas com deficiência, permitindo que leitores de tela e outros dispositivos interpretem corretamente a estrutura e o significado do conteúdo.
