Esse é nosso primeiro semestre

E as materias foram:

Linguagem de programação

Aqui studamos HTML,CSS e Javascript. Alem de logica e conceitos como DOM e semantica

Automação

Aqui estudamos conceitos de robotica e eletronica. Tambem vimos linguagem C e arduino

Redes de computadores

Aqui estudamos varios conceitos de redes como modelos, camadas e potocolos. tambem fizemos alguns exercicios de configuração

Banco de dados

Aqui estudamos sobre modelagem de banco de dados e fizemos o uso da linguagem SQL para modelalos fisicamente

Ciencia de dados

Aqui estudamos coisas como amostragem, população entre outra formas de analisar dados

Resumo do semestre

Durante o primeiro semestre do curso de Tecnólogo em Análise e Desenvolvimento de Sistemas, os alunos foram introduzidos a uma variedade de disciplinas que forneceram uma base sólida para a compreensão dos principais conceitos e habilidades necessários na área de tecnologia. Nesse período, foram abordadas as seguintes matérias: linguagem de programação, automação, redes de computadores, banco de dados e ciência de dados. Ao clicar nos cards a cima podera ver um pouco sobre cada meateria em especifico e alguns projetos realizados em aula para assim aprimorar nossos conhecimento e fixarmos os mesmos.
